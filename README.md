## 🧠 Vocabulary Builder Game (Python)

A command-line vocabulary training game built in Python to help improve English vocabulary through interactive multiple-choice questions.

---

## 🎮 Features
- Difficulty levels (**Easy / Medium / Hard**)
- Multiple-choice quiz system  
- Randomized questions and answer options  
- Lives system (game ends after 3 mistakes)  
- Score tracking with percentage feedback  
- High score saving (persistent across sessions)  
- Review system for incorrect answers  

---

## 🚀 How to Run
1. Make sure Python is installed  
2. Ensure `words.json` is in the same directory as the script  
3. Run the program:

```bash
python smartVocab.py
```
